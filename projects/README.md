# Project Showcase Entries

One folder per selected project, added by organizers after the event.

## Entry format

```
projects/
  <team-slug>/
    README.md       # the write-up (template below)
    screenshot.png  # one hero image, ≤ 1 MB
```

## Write-up template

```markdown
# <Project Name>

> <Award won> — Abu Dhabi AI PropTech Challenge

**Team:** <team name> (<members>)
**Track:** <track>

## What it does
2–3 sentences.

## Why it won
1–2 sentences from the judges' perspective.

## Links
- Repo: <url>
- Demo: <url>
- Slides: <url>

![screenshot](screenshot.png)
```

## Rules

- Entries are added with the team's consent; teams keep ownership and license of their work.
- Keep write-ups factual — describe what was demoed, not the roadmap.
- No personal contact details beyond GitHub handles.
